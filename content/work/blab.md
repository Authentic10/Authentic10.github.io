<div style="text-align:justify;">Blab est un réseau social. Toutes les fonctionnalités n'ont pas été implémentées. Le but était d'apprendre la programmation mobile et de voir comment ces types d'applications fonctionnent.</div><br/>

<div style="text-align:justify;">Les utilisateurs peuvent créer des comptes en utilisant leur numéros de téléphone, se suivrent et ajouter de nouveaux posts. Les interactions tels que les commentaires et messages n'ont pas été implémentées.</div><br/>

<div style="text-align:justify;">Ce projet a été créé avec Java Android et Firebase. Le code source est disponible sur Github : <a href="https://github.com/Authentic10/Blab" target="blank">Blab</a></div>

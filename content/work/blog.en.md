title: "Blog App"
description: "Blog app that users can use to create, comment posts and follow each other"
---


<div style="text-align:justify;">This project is a web application that let users to create posts. Users can follow each other and react to the posts of the users they are following by adding comments. </div> <br/>

<div style="text-align:justify;">This project was built with Java Enterprise Edition, Bootstrap, JQuery and MySQL. The source code is available on Github : <a href="https://github.com/Authentic10/Blog" target="blank">Blog App</a></div>

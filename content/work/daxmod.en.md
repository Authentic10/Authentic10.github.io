<div style="text-align:justify;">Daxmod is a toolbox that allows engineers and researchers to perform text classification experiments. It offers modules for data analysis, feature extraction, feature selection, model building and model persistence.</div> <br/>

<div style="text-align:justify;">Daxmod was built with Python. The source code is available on Github : <a href="https://github.com/Authentic10/daxmod" target="blank">Daxmod</a></div>

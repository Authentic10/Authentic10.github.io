---
title: "Daxmod"
description: "Daxmod est une boîte à outils développée pour la classification de texte"
---

<div style="text-align:justify;">Daxmod est un outil qui permet aux ingénieurs et chercheurs d'effectuer des expérimentations sur la classification de texte. Il fournit des modules pour l'analyse de données, l'extraction et la sélection des caractéristiques, la création des modèles d'apprentissage et la persistence des modèles.</div> <br/>

<div style="text-align:justify;">Daxmod a été développé avec Python. Le code source est disponible sur Github : <a href="https://github.com/Authentic10/daxmod" target="blank">Daxmod</a></div>

<h3 id="state">2. Visualization based on states </h3>


- Question 4: Which state has the most deaths ?


![deaths_by_states](/graph-4.png)

<br/>
<div style="text-align:justify;">We can see in the graph above that California and Texas are the states where most deaths occured. This is not surprising because they have the largest populations respectively. It is the same thing for Florida and other states as well so according the dataset, we can say that there is a correlation between the population and the number of deaths. Let's verify this hypothesis with the next graph. </div>

- Question 5 : Is there a correlation between the population and the number of deaths ?

![correlation_population_deaths](/graph-5.png)

<br/>
<div style="text-align:justify;">The graph shows that there is a strong correlation between the population and the number of deaths. This confirms our hypothesis.</div>

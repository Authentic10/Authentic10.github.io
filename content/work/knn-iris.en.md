<div style="text-align:justify;">In this project, we implemented the KNN algorithm from scratch on the <a href='https://archive.ics.uci.edu/ml/datasets/Iris/' target='_blank'>iris</a> dataset. We implemented the following features : </div>

<ul>
    <li>Euclidean distance computing </li>
    <li>Train et test split</li>
    <li>Computing of metrics like <strong>precision</strong> and <strong>recall</strong> </li>
    <li>Confusion matrix<li>
    <li>Cross Validation</li>
</ul>

<div style="text-align:justify;">The project has been implemented in Java and it is available on Github : <a href="https://github.com/Authentic10/kPPV" target="blank">KNN</a></div>

Ask



Ask est une application web créée pour les demandes de permission en entreprise. Les utilisateurs peuvent effectuer des demandes en précisant des informations comme le motif, la durée, etc. Après une demande de permission, l'application notifie le supérieur de l'utilisateur en envoyant un mail. Le supérieur peut refuser ou accepter la demande. Une fois, la décision prise, l'utilisateur reçoie un mail pour lui indiquer la décision de sa demande.

Ce projet a été créé avec Java EE, Bootstrp et MySQL.

Blab


Inscription Accueil Profil

Blab est un réseau social. Toutes les fonctionnalités n'ont pas été implémentées. Le but était d'apprendre la programmation mobile et de voir comment ces types d'applications fonctionnent.

Les utilisateurs peuvent créer des comptes en utilisant leur numéros de téléphone, se suivrent et ajouter de nouveaux posts. Les interactions tels que les commentaires et messages n'ont pas été implémentées.

Ce projet a été créé avec Java Android et Firebase. Le code source est disponible sur Github : Blab
